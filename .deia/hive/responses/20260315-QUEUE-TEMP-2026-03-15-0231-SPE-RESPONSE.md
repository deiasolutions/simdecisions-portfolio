### Root Cause Analysis
The regressions were caused by event type naming inconsistency:

1. **Symptom:** Smoke tests querying for `"STORAGE_WRITE"` events found zero results, even after writes succeeded
2. **Root cause:** Transport layer emitted events with dot notation (`"storage.write"`) but tests queried for uppercase underscore format (`"STORAGE_WRITE"`)
3. **Inconsistency in codebase:** Some parts use dots (`"llm.route"`), others use uppercase underscores (`"NODE_ANNOUNCED"`, `"LLM_CALL"`, `"SYNC_STARTED"`)
4. **Fix:** Standardized storage events to uppercase underscore format to match the majority pattern
