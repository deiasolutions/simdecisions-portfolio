## Root Cause Analysis
The chat.egg.md uses a tree-browser pane with `adapter: "chat-history"`. The chatHistoryAdapter loads conversations from chatApi and groups them by date (Today, Yesterday, This Week, Older). The TreeBrowser component requires groups to be explicitly expanded to show their children. The treeBrowserAdapter.tsx component has logic (lines 147-159) to auto-expand top-level groups for adapters in the AUTO_EXPAND_ADAPTERS set. The 'chat-history' adapter was missing from this set, causing date groups to appear collapsed with no visible conversations underneath.
