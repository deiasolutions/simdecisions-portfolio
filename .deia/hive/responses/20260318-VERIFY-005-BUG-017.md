### 1. OAuth Token Extraction Logic (App.tsx)

**Lines 24-49** — `extractTokenFromUrl()` function exists and is fully implemented:

```typescript
function extractTokenFromUrl(): void {
  const params = new URLSearchParams(window.location.search)
  const token = params.get('token')
  if (!token) return

  try {
    const parts = token.split('.')
    if (parts.length !== 3) return
    const payload = JSON.parse(base64UrlDecode(parts[1]))
    const user = {
      id: payload.sub || payload.id || '',
      email: payload.email || '',
      display_name: payload.display_name || '',
    }
    setToken(token)
    setUser(user)
  } catch {
    // Malformed JWT — ignore
  }

  // Clean token from URL regardless of success
  window.history.replaceState({}, '', window.location.pathname)
}

// Run before React renders
extractTokenFromUrl()
```

**What this does:**
1. Extracts `?token=` from URL query params
2. Validates JWT structure (3 parts separated by dots)
3. Decodes JWT payload using `base64UrlDecode()`
4. Extracts user info (`sub`, `email`, `display_name`)
5. Saves token and user to `authStore` via `setToken()` and `setUser()`
6. Cleans the URL with `window.history.replaceState()` to remove the token from browser history
7. Runs at module load time (line 49) before any React renders
