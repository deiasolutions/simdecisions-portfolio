## Architecture Notes

- **In-memory storage:** Uses Python dicts and lists (no filesystem I/O)
- **Mutability:** Spec content is mutable — `append_section()` modifies object in place
- **Event tracking:** Append-only event list for audit trail
- **Defensive copy:** `list_specs()` returns `.copy()` to prevent external mutation
- **Dependency resolution:** Substring matching (`any(dep in done_id for done_id in done_ids)`) allows partial ID matching
- **ABC inheritance:** Properly inherits from `PipelineStore`; class is a concrete implementation, not stub
