## What Was Done
- Created `useTrayGestures.ts` hook that handles both swipe-to-minimize and long-press-to-menu on the same touch lifecycle
  - Swipe down >60px on tray minimizes keyboard to pill
  - Long-press (600ms) on tray opens options menu
  - Guards against triggering on key elements (data-key attribute check)
  - Cancels long-press timer on ANY movement or touch end
  - Emits RTD events: `keyboard:minimized` and `keyboard:restored`
- Created `useOptionsMenu.ts` hook that manages menu state and formats 5 menu items
  - Readout position: cycles top → bottom → hidden → top
  - Return lock: toggles on/off
  - Fn lock: toggles on/off
  - F-row visibility: toggles visible/hidden
  - Buffer window: cycles 14 → 20 → 8 → 14 chars
  - Menu closes after each action
- Created `OptionsMenu.tsx` presentational component
  - Absolute-positioned panel with backdrop for outside-click detection
  - Renders 5 menu items + close link
  - Theme-aware styling (dark/light mode)
  - Accent-colored close link
- Created `usePhysicalKeyboard.ts` hook for physical keyboard detection
  - Listens for trusted keydown events
  - Auto-minimizes on-screen keyboard when physical keyboard detected
  - Emits RTD: `keyboard:physical-detected`
  - Calls onMinimize only once per detection session
- Updated `SCKeyboard.tsx` to integrate all Wave C features
  - Migrated inline `minimized` and `showMenu` state to hooks
  - Added touch handlers (onTouchStart/Move/End) to keyboard surface div in both layouts
  - Added right-click handler (onContextMenu) to open menu on desktop
  - Rendered `OptionsMenu` component in both compact and full layouts
  - Integrated physical keyboard detection hook
- Updated `useFnLayer.ts` to expose `setFnLocked` for options menu control
- Created comprehensive tests for all three hooks (24 total tests)
  - `trayGestures.test.ts`: 8 tests covering swipe, long-press, guards, and cancellation
  - `optionsMenu.test.ts`: 11 tests covering all 5 menu items, cycles, toggles, and menu state
  - `physicalKeyboard.test.ts`: 5 tests covering detection, event listener cleanup
