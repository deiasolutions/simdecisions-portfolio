## Smoke Test Checklist

### Visual Checks
- [ ] **All files expanded by default**
  - Open page with 3-file diff
  - Verify all file bodies are visible
  - Verify all chevrons show ▼

- [ ] **Click file header to collapse**
  - Click first file header
  - Verify chevron changes to ▶
  - Verify file body hidden (height: 0)
  - Verify smooth 200ms animation

- [ ] **Click collapsed file header to expand**
  - Click collapsed file header again
  - Verify chevron changes to ▼
  - Verify file body visible
  - Verify smooth 200ms animation

### Keyboard Shortcuts
- [ ] **Ctrl+E expands all files**
  - Collapse 2-3 files manually
  - Press Ctrl+E
  - Verify all files expand
  - Verify all chevrons show ▼

- [ ] **Ctrl+Shift+E collapses all files**
  - Press Ctrl+Shift+E
  - Verify all files collapse
  - Verify all chevrons show ▶

### Persistence
- [ ] **localStorage saves collapsed state**
  - Collapse first file
  - Open DevTools → Application → Local Storage
  - Verify key `sd:diff_viewer_collapsed` exists
  - Verify value is `{"src/app.ts": true}` (or similar)

- [ ] **Reload restores collapsed state**
  - Collapse 2nd and 3rd files
  - Reload page (F5)
  - Verify collapsed files remain collapsed
  - Verify first file remains expanded

### Accessibility
- [ ] **File header is button with aria-expanded**
  - Inspect file header element
  - Verify it's a `<button>` element
  - Verify `aria-expanded="true"` when expanded
  - Verify `aria-expanded="false"` when collapsed

- [ ] **Keyboard navigation works**
  - Tab to file header button
  - Press Enter → file collapses
  - Press Space → file expands

### CSS Variables
- [ ] **No hardcoded colors**
  - Inspect file header styles
  - Verify all colors use `var(--sd-*)`
  - Check hover state uses `var(--sd-surface-hover)`
  - Check focus outline uses `var(--sd-accent)`
