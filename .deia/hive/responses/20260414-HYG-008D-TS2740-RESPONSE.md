## Implementation Details

Each test file now includes a helper function that creates complete `EggUiConfig` objects:

```typescript
const createTestUiConfig = (overrides?: Partial<EggUiConfig>): EggUiConfig => ({
  displayName: 'TestApp',
  menuBar: false,
  shellTabBar: false,
  statusBar: true,
  theme: 'default',
  titleBar: true,
  actionBar: true,
  leftRail: false,
  leftSurface: 'collapsed',
  leftSurfaceWidth: 280,
  leftSurfaceDefault: null,
  rightDock: 'collapsed',
  rightDockWidth: 320,
  commandPalette: true,
  ...overrides,
});
```

This approach:
- Provides type-safe defaults for all required properties
- Allows tests to override only what they need
- Makes test intent clear (e.g., `createTestUiConfig({ menuBar: true })`)
- Prevents future TS2740 errors if new required properties are added
