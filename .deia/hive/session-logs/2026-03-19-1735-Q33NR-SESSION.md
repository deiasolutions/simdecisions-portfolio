### What Was Found (8 interconnected systems):

**1. ShiftCenter Patois** (`simdecisions-2/src/services/frank/prompts/patois.md` + `.deia/hive/coordination/patois.md`)
- Machine-readable language protocol. Vowel-substituted machine names (Fr@nk, 6laude, Q33N, b33). Seven core roles. Three Currencies. Principles: ABCDEFG, NOKINGS, "Understand loose, speak tight."

**2. Dialect Layer Architecture** (`frank/dialectLoader.ts` + `prompts/envelope.md`)
- Composable system prompt dialects: Patois (identity) + Envelope (response format) always active. Additional dialects (IR-Generation, Simulation, Co-Author) loaded contextually. `composeSystemPrompt()` assembles at request time.

**3. Vocabulary Theme System / Dialect Packs** (`src/data/dialectPacks.ts` + coordination docs)
- User-selectable terminology for the SAME canonical primitives. Four themes: Developer (Task/Queue/Fork), PMP (Work Package/Decision Gate/Milestone), Engineer (Process Step/Branch Point/Quality Gate), Corporate (Agent/Worker/Coordinator).
- `resolveTerm(packId, canonical)` translates. Cascade: active pack → Developer → canonical key.
- 5-phase implementation plan was APPROVED: migration script → theme layer → primitive IR → identity/eggs → training corpus → QA.

**4. Fr@nk Input Modes** (`.deia/hive/coordination/2026-03-08-FRANK-INPUT-MODES-v2.md`)
- Text mode, Formatted text (`python:`, `bash:` prefixed), Natural language command (`/ ` prefix), Slash commands (`/clear`, `/help`). Co-Author overlay (🤝 toggle, diff-based rewriting).

**5. RAG EGGs for Personality Modes** (`ADR-FRANK-001-EGG-EXTENSION-ARCHITECTURE.md`)
- Extended capabilities (planning mode, strict unix mode, domain personas) loaded as RAG EGGs at runtime. Not hardcoded. EGG bundles include systemPromptOverlay, ragContext, permissions, UI config.

**6. Domain Dialect Compilers** (`canonical/docs/adr-013-domain-dialect-compilers.egg.md`)
- Plugins translating domain languages (BPMN, SBML, Terraform, custom) into PHASE-IR. Lossless round-trip via PIE packages. Plugin protocol: `parse()` → `transform()` → `export()`.

**7. Primitive Role Architecture** (`.deia/hive/coordination/2026-03-04-Q33N-PRIMITIVE-DIALECT-ARCHITECTURE.md`)
- 19 canonical primitives below all dialects. User A says "bee", User B says "agent", User C says "b33" → all normalize to `llm_worker`. CSS analogy: primitive = `var(--sd-role)`, dialect = theme resolving it.

**8. Settings Store** (`src/stores/settingsStore.ts`)
