## Specification

Write the EGG file for the apps home page. Must follow schema_version 3.

### Frontmatter

```yaml
---
egg: apps-home
version: 0.1.0
schema_version: 3
displayName: App Directory
description: Browse and launch ShiftCenter EGGs.
author: daaaave-atx
defaultRoute: /apps
license: MIT
_stub: false
---
```

### Layout block

Single pane. The `apps-home` appType renders the full card grid.

```json
{
  "type": "pane",
  "nodeId": "apps-directory",
  "appType": "apps-home",
  "label": "App Directory",
  "config": {
    "sections": ["core", "productivity", "platform"],
    "showSearch": true
  }
}
```

**IMPORTANT:** Use `"type": "pane"`, NOT `"type": "app"`. The shell's `eggToShell.ts` parser expects `"pane"`.

### UI block

Minimal chrome. No menu bar, no tab bar, no activity bar.

```json
{
  "hideMenuBar": true,
  "hideStatusBar": false,
  "hideTabBar": true,
  "hideActivityBar": true
}
```

### Tabs block

```json
[
  { "id": "tab-apps", "eggId": "apps-home", "label": "Apps", "active": true }
]
```

### Permissions block

```json
{
  "storage": { "localStorage": true, "sessionStorage": false },
  "network": { "allowedDomains": ["localhost", "hivenode.railway.app"] },
  "bus_emit": ["egg:inflate"],
  "bus_receive": []
}
```

### Settings block

```json
{
  "defaultView": "grid"
}
```

Do NOT include: modes, away, startup, commands, or prompt blocks. This is a static directory page.
